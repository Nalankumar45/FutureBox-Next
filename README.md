# FutureBox - AI-Powered Innovation Platform

Transform ideas into innovation reality with AI-powered workflows, ML analytics, and intelligent automation.

## Tech Stack

- **Framework:** Next.js 15+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui (Radix UI primitives)
- **State Management:** TanStack Query
- **Database:** PostgreSQL (via Drizzle ORM)

## 8 Powerful Modules

1. **IdeaHub** - Submit, manage, and enhance ideas with AI-powered suggestions
2. **CollabSpace** - Collaborate through discussions and team connections
3. **Impact Engine** - Evaluate ideas with automated scoring and analytics
4. **ActionZone** - Track workflow stages with AI-powered alerts
5. **Recognition Arena** - Gamification with points and badges
6. **Profile & Insights** - Analytics dashboards and recommendations
7. **Alert Center** - Smart notifications with sentiment analysis
8. **Admin & Policy** - Governance tools with role-based access

## Getting Started

### Development

```bash
npm run dev
```

The application will start on port 5000.

### Build

```bash
npm run build
```

### Production

```bash
npm run start
```

## Design System

FutureBox follows a modern enterprise dashboard design approach inspired by Linear, Notion, and Vercel, emphasizing clarity, scalability, and professional polish.

### Key Principles

- **Information Clarity** - Dense data with clear hierarchy
- **Intelligent Simplicity** - Complex AI features made accessible
- **Enterprise Polish** - Professional aesthetic
- **Modular Consistency** - Unified design across all modules
